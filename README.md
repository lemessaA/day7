# Agentic Coordination Graph

A simple demo of multi-agent workflows using **LangGraph**.

## Features

* Functional API for data processing
* Graph API for branching agent workflows
* Typed shared state for clarity and safety

## Run

```bash
python main.py
```

## Structure

* `main.py` — defines tasks, agents, and graph
* `README.md` — project overview

## Purpose

Demonstrates explicit control flow and modular agent orchestration.
